# Surprice x Voyage Rent - Co-Branded Logo Package

## Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Surprice Red | `#C8102E` | "Sur" text, Rent text, icon top, divider |
| Surprice Navy | `#1B2A4A` | "price" text, Voyage text, icon bottom, taglines |
| White | `#FFFFFF` | Background, icon highlights |
| Dark BG Red | `#E83E3E` | Red on dark backgrounds |

## Font

Primary: **Nunito** (Bold 800) - rounded sans-serif
Fallbacks: Varela Round, Arial Rounded MT Bold, Arial

## File Overview

### SVG (Vector - for print, signage, large format)

| File | Use Case |
|------|----------|
| `surprice-voyagerent-full-horizontal.svg` | Main logo with icon - letterheads, website header |
| `surprice-voyagerent-text-horizontal.svg` | Text-only horizontal - compact spaces |
| `surprice-voyagerent-car-sticker.svg` | Car sticker with white background (80x20cm) |
| `surprice-voyagerent-white-bg-sticker.svg` | Large format sticker (100x30cm) |
| `surprice-voyagerent-stacked.svg` | Vertical/square layout - social media, signs |
| `surprice-voyagerent-dark-bg.svg` | For dark vehicle colors or dark backgrounds |

### PNG (Raster - for digital, web, email)

Located in `png/` folder with size variants:
- `*-1x.png` - Standard resolution (web/digital)
- `*-2x.png` - Retina/high-DPI screens
- `*-4x.png` / `*-print.png` - High-res for print

## Recommended Sizes

| Application | File to Use | Print Size |
|-------------|-------------|------------|
| Car rear sticker | `car-sticker` or `white-bg-sticker` | 60-80cm wide |
| Signage / banner | `full-horizontal` SVG | Scale as needed |
| T-shirt (chest) | `stacked` SVG | 25-30cm wide |
| T-shirt (back) | `full-horizontal` SVG | 30-35cm wide |
| Business card | `text-horizontal` SVG | Full width |
| Social media avatar | `stacked` PNG 2x | 400x400px crop |
| Website | `full-horizontal` PNG 2x | As-is |

## Print Notes

- **Always use SVG files for professional printing** - they scale to any size without quality loss
- For car stickers: use `car-sticker` variant (includes rounded corners and cut marks)
- Minimum clear space: 10% of logo height on all sides
- Do not distort, rotate, or change the color scheme
- For Pantone matching: Red ~ Pantone 186 C, Navy ~ Pantone 289 C

## Font Installation

For the best rendering, install **Nunito** font from Google Fonts:
