# VoyageRent — Финансовая модель бизнеса по аренде автомобилей

## Полное руководство по использованию финансовой модели

---

## 1. СТРУКТУРА МОДЕЛИ

Модель состоит из 8 листов, каждый выполняет свою роль:

| # | Лист | Назначение |
|---|------|-----------|
| 1 | **INPUTS** | Все исходные данные — единственный лист для ввода |
| 2 | **UNIT ECONOMICS** | Экономика одного автомобиля |
| 3 | **FLEET PERFORMANCE** | Показатели всего автопарка |
| 4 | **REVENUE FORECAST** | Прогноз доходов на 12 месяцев с сезонностью |
| 5 | **P&L** | Отчёт о прибылях и убытках (годовой) |
| 6 | **CASH FLOW** | Движение денежных средств по месяцам |
| 7 | **DASHBOARD** | Визуальная панель с KPI и графиками |
| 8 | **SCENARIOS** | Сценарный анализ (pessimistic / base / optimistic) |

**Принцип работы:** Все данные вводятся ТОЛЬКО на листе INPUTS. Остальные 7 листов рассчитываются автоматически через формулы, ссылающиеся на INPUTS.

## 2. ЛИСТ INPUTS — Подробное описание параметров

### 2.1 Автопарк (Fleet)

| Ячейка | Параметр | Пример | Что означает |
|--------|----------|--------|-------------|
| C6 | Количество автомобилей | 100 | Текущий размер флота |
| C7 | Стоимость автомобиля | 18 000 € | Средняя закупочная цена |
| C8 | Срок амортизации | 5 лет | За сколько лет списывается стоимость |
| C9 | Остаточная стоимость | 20% | Сколько % от цены стоит авто после амортизации |

### 2.2 Доходы (Revenue)

| Ячейка | Параметр | Пример | Что означает |
|--------|----------|--------|-------------|
| C12 | Цена аренды в день | 45 € | Средневзвешенная цена по всем классам |
| C13 | Средняя продолжительность аренды | 5 дней | Типичная аренда |
| C14 | Загрузка автопарка | 65% | Какой % времени авто сдано |
| C15 | Количество аренд в месяц (на 1 авто) | 4 | Сколько раз авто сдаётся за месяц |
| C16 | Комиссия агрегаторов | 15% | Сколько забирают Booking, Rentalcars и т.д. |

### 2.3 Расходы на автомобиль (Per Vehicle Costs)

| Ячейка | Параметр | Пример | Что означает |
|--------|----------|--------|-------------|
| C19 | Страховка | 120 €/мес | Полная страховка + КАСКО |

| C22 | Топливо / подготовка | 40 €/мес | Подготовка авто, мойка |

| Ячейка | Параметр | Пример | Что означает |

## 3. КЛЮЧЕВЫЕ ФОРМУЛЫ

### 3.1 UNIT ECONOMICS — Формулы экономики одного автомобиля

```
Доход за 1 аренду          = Цена/день × Дни аренды
                            = INPUTS!C12 × INPUTS!C13

Месячный доход (gross)      = Доход за аренду × Аренд в месяц
                            = (C12 × C13) × C15

Комиссия агрегаторов        = Месячный gross × % комиссии
                            = Месячный_gross × C16

Месячный доход (net)        = Gross - Комиссия

Амортизация (месяц)         = Стоимость × (1 - Ост.%) / (Срок × 12)
                            = C7 × (1 - C9) / (C8 × 12)

Итого расходы на авто/мес   = Амортизация + Страховка + Ремонт + Парковка
                              + Топливо + ITV/12

Валовая прибыль (месяц)     = Доход net - Расходы на авто

Срок окупаемости            = Чистая стоимость / Валовая прибыль в месяц
                            = C7 × (1 - C9) / Валовая_прибыль_мес

ROI (годовой)               = Годовая прибыль / Стоимость авто
```

### 3.2 FLEET PERFORMANCE — Формулы автопарка

```
Дней аренды на авто/мес     = 30 × Загрузка
Аренд в день (флот)         = (Аренд на авто/мес / 30) × Кол-во авто
Месячный доход флота (net)  = Доход_за_аренду × Аренд_в_мес × Флот × (1 - Комиссия)
Валовая прибыль флота       = Доход_net - Расходы_на_авто × Флот
```

### 3.3 REVENUE FORECAST — Формулы прогноза

```
Сезонный коэффициент        = IF(месяц в [6-10], Коэф_высокий, Коэф_низкий)
Загрузка с сезоном          = Базовая_загрузка × Сезонный_коэффициент
Кол-во авто (месяц i)       = ROUND(Флот × (1 + Рост × i/12), 0)
Дней аренды (флот/мес)      = 30 × Загрузка_сезон × Авто_в_месяце
Выручка gross               = Дней_аренды × Цена/день
EBITDA                      = Выручка_net - Все_расходы
```

### 3.4 P&L — Формулы отчёта о прибылях

```
Выручка net                 = Gross - Комиссии
COGS (себестоимость)        = Амортизация + Страховка + Ремонт + Парковка
                              + Топливо + ITV (× Флот × 12)
Валовая прибыль             = Выручка net - COGS
OPEX                        = ФОТ + Аренда + Маркетинг + Софт + Связь + Прочие
EBITDA                      = Валовая прибыль - OPEX
Чистая прибыль              = EBITDA - Налог_на_прибыль
```

## 4. СВЯЗИ МЕЖДУ ЛИСТАМИ

```
INPUTS ──────► UNIT ECONOMICS ──────► FLEET PERFORMANCE
   │                │                        │
   │                ▼                        ▼
   ├──────► REVENUE FORECAST ◄──────────────┘
   │                │
   │                ▼
   ├──────────► P&L ◄───────────── FLEET PERFORMANCE
   │                │
   │                ▼
   ├──────► CASH FLOW ◄──────────── REVENUE FORECAST
   │
   ├──────► SCENARIOS
   │
   └──────► DASHBOARD ◄──── все листы (KPI + графики)
```

Все стрелки идут от INPUTS. Ни один лист не содержит "жёстко зашитых" чисел — всё рассчитывается из входных параметров.

## 6. КАК ПРАВИЛЬНО ЗАПОЛНЯТЬ МОДЕЛЬ

### Обязательные данные (минимум для работы модели):
1. **Количество автомобилей** — точное число вашего флота
2. **Цена аренды** — посчитайте средневзвешенную по классам
3. **Загрузка** — реальный % из вашей CRM/системы бронирования
4. **Стоимость автомобиля** — средняя закупочная цена
5. **Расходы на авто** — реальные данные из бухгалтерии
6. **ФОТ и overhead** — из бухгалтерского учёта

### Как рассчитать средневзвешенную цену аренды:
```
Пример:
- Эконом (30 авто): 30 €/день
- Комфорт (50 авто): 45 €/день
- Премиум (20 авто): 70 €/день

Средневзвешенная = (30×30 + 50×45 + 20×70) / 100 = 43.50 €/день
```

### Как определить загрузку:
```
Загрузка = Дни в аренде / Общие доступные дни

Пример: 100 авто × 30 дней = 3000 доступных дней
Если сдано 1950 дней → Загрузка = 1950/3000 = 65%
```

## 9. КАК ИСПОЛЬЗОВАТЬ МОДЕЛЬ ДЛЯ ПЛАНИРОВАНИЯ

### Планирование года:
1. Заполните INPUTS реальными данными
2. Посмотрите REVENUE FORECAST — годовой прогноз
3. Откройте P&L — ожидаемая прибыль за год
4. Проверьте CASH FLOW — хватит ли денег
5. Используйте SCENARIOS — что если загрузка упадёт?

### Планирование квартала:
1. Смотрите квартальную сводку на REVENUE FORECAST (строки 28-32)
2. Сравнивайте Q1-Q4 с учётом сезонности
3. Планируйте маркетинговый бюджет под низкий сезон

### Планирование месяца:
1. REVENUE FORECAST — столбец нужного месяца
2. Сколько аренд нужно? → FLEET PERFORMANCE
3. Какая выручка ожидается? → с учётом сезонного коэффициента

### Сколько аренд нужно в день:
```
Целевая выручка (мес) / Доход за 1 аренду / 30 дней = Аренд в день

Пример: 150 000 € / 225 € / 30 = 22.2 аренды в день
```

### Сколько авто нужно для целевого оборота:
```
Целевой оборот (год) / Годовой доход на 1 авто = Нужное кол-во авто

Пример: 2 000 000 € / 18 360 € = 109 автомобилей
```

## 10. ТОП-10 KPI ДЛЯ БИЗНЕСА АРЕНДЫ АВТОМОБИЛЕЙ

### Критически важные (следить постоянно):

| # | KPI | Формула | Целевое значение | Важность |
|---|-----|---------|-----------------|----------|
| 1 | **Загрузка автопарка** | Дни в аренде / Доступные дни | > 65% | Главный драйвер дохода |
| 2 | **Revenue per Available Car Day (RevPACD)** | Выручка / (Авто × Дни) | > 30 € | Аналог RevPAR в отелях |
| 3 | **EBITDA маржа** | EBITDA / Выручка net | > 25% | Эффективность бизнеса |
| 4 | **Средний доход на автомобиль (мес)** | Выручка / Авто / 12 | > 1 000 € net | Эффективность флота |
| 5 | **Срок окупаемости автомобиля** | Стоимость / Прибыль в мес | < 30 мес | Возврат инвестиций |

### Важные (проверять регулярно):

| # | KPI | Формула | Целевое значение | Важность |
|---|-----|---------|-----------------|----------|
| 6 | **Средняя продолжительность аренды** | Сумма дней / Кол-во аренд | 4-7 дней | Оптимальный диапазон |
| 7 | **Доля прямых бронирований** | Прямые / Все бронирования | > 40% | Снижение комиссий |
| 8 | **Расходы на ремонт / авто** | Общий ремонт / Флот | < 100 €/мес | Контроль затрат |
| 9 | **Стоимость привлечения клиента (CAC)** | Маркетинг / Новые клиенты | < 30 € | Эффективность маркетинга |
| 10 | **ROI на автомобиль (годовой)** | Год. прибыль / Стоимость | > 25% | Инвестиционная привлекательность |

### Приоритеты KPI:

**Самые важные** (80% внимания):
1. Загрузка — если авто стоят, вы теряете деньги
2. RevPACD — комбинирует цену и загрузку
3. EBITDA маржа — реальная эффективность

**Важные** (15% внимания):
4. Доход на авто
5. Срок окупаемости
6. Доля прямых бронирований

**Дополнительные** (5% внимания):
7-10 — мониторинг трендов

## 11. КАК ИСПОЛЬЗОВАТЬ МОДЕЛЬ ДЛЯ РОСТА КОМПАНИИ

### Стратегия 1: Оптимизация загрузки
1. В INPUTS измените загрузку с 65% на 75%
2. Посмотрите, как изменилась прибыль (P&L)
3. Рассчитайте: сколько стоит увеличить загрузку на 10%?
4. Сравните затраты на маркетинг vs. прирост прибыли

### Стратегия 2: Расширение флота
1. В INPUTS увеличьте количество автомобилей
2. Проверьте CASH FLOW — хватит ли денег на покупку?
3. Используйте SCENARIOS — при какой загрузке новые авто окупаются?

### Стратегия 3: Увеличение цены
1. В INPUTS поднимите цену аренды
2. Но учтите: загрузка может упасть!
3. Используйте SCENARIOS: цена +10€, загрузка -5% — что лучше?

### Стратегия 4: Снижение комиссий
1. Увеличьте долю прямых бронирований
2. В INPUTS снизьте комиссию агрегаторов
3. Перенаправьте часть экономии в маркетинг собственного сайта

### Стратегия 5: Оптимизация расходов
1. Пересмотрите страховку — возможно, есть лучшие условия?
2. Оптимизируйте ремонт — профилактика дешевле починки
3. Сравните текущие расходы с бенчмарками отрасли
